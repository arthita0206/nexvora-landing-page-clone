# Nexvora Digital Landing Page Clone

A responsive frontend clone of the Nexvora Digital landing page built using HTML, CSS, and JavaScript.

## 📖 Overview

This project recreates the Nexvora Digital landing page with a modern, responsive design and a fully functional dark/light theme toggle.

The website is designed to provide a smooth user experience across desktop, tablet, and mobile devices while maintaining clean and scalable code.

---

## ✨ Features

- Responsive Landing Page
- Modern UI Design
- Mobile-Friendly Navigation
- Interactive Hero Section
- Core Services Section
- Process Timeline Section
- Pricing / Investment Plans
- Portfolio Showcase
- Team Section
- Contact CTA Section
- Professional Footer
- Dark Mode / Light Mode Toggle
- Theme Persistence using Local Storage
- Smooth Hover Effects & Transitions

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📱 Responsive Design

The website is optimized for:

- Desktop
- Tablet
- Mobile Devices

---

## 🎨 Theme Toggle

Users can switch between:

- Dark Theme
- Light Theme

The selected theme is saved using Local Storage and remains active after page refresh.
